# Todo Application

A simple web application that helps users organize and track their daily tasks.

## Overview

This application allows users to create an account, securely log in, and manage their personal todo list. Users can add tasks, monitor their progress throughout the day, and mark tasks as completed or remove them when no longer needed.

## Features

* User Registration and Authentication

  * Create a new account.
  * Secure login using JWT-based authentication.

* Authorization System

  * Each user can only access their own tasks.
  * Protected routes ensure that users cannot view, modify, or delete another user's todos.

* Todo Management

  * Create new todo items.
  * View all personal todos.
  * Update existing tasks.
  * Mark tasks as completed.
  * Delete completed or unwanted tasks.

* Secure Backend

  * Password hashing for user credentials.
  * Token-based authentication.
  * User-specific data isolation.

## Tech Stack

* FastAPI
* SQLAlchemy
* PostgreSQL
* JWT Authentication
* Python
* jinja2
* html,css
* js
  

## Project Goal

The goal of this project is to practice building a complete backend application with authentication with simple frontend , authorization, database integration, and RESTful API development while following secure software development practices.
